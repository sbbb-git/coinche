# Suivi des positions SEO — Coincheur

> Le site est neuf : les positions mettent quelques semaines à apparaître. Voici
> comment suivre, quoi regarder, et les mots-clés à tracker.

## 1. Google Search Console (gratuit, la source de vérité)
- **Performances** → onglet *Requêtes* : impressions, clics, **position moyenne**, CTR.
  - Trie par impressions pour voir sur quoi tu apparais déjà.
  - Repère les requêtes en position **5–20** (« presque 1ʳᵉ page ») : ce sont les
    pages à renforcer en priorité (ajouter du contenu, des liens internes).
- **Performances** → *Pages* : quelles URLs rapportent. Double les efforts dessus.
- **Indexation** → *Pages* : vérifier que les ~250 URLs sont bien indexées
  (resoumettre le sitemap si besoin).
- Cadence conseillée : **1 fois/semaine**, 10 min.

## 2. Rank tracker (optionnel, plus précis par mot-clé)
Ahrefs (déjà sur le site) / SE Ranking / Ubersuggest : ajouter le **projet
coincheur.fr** + la liste de mots-clés ci-dessous, marché **France (FR)**.

## 3. Mots-clés à suivre (par priorité)

### Tête / marque
- coincheur

## 4. Que faire selon les résultats
- **Position 5–15** sur un mot-clé : enrichir la page (longueur, FAQ, exemples),
  ajouter 2–3 liens internes depuis des pages proches, viser le mot-clé dans le
  titre/H1.
- **Beaucoup d'impressions, peu de clics** : retravailler le **title** et la
  **meta description** (plus accrocheurs) — c'est le levier CTR.
- **Page non indexée** : vérifier le sitemap, demander l'indexation dans GSC.
- **Rien après 6–8 semaines** sur un thème : renforcer le netlinking
  (cf. `NETLINKING.md`) et le maillage interne vers la page.

## 5. Objectif réaliste
- Semaines 1–4 : indexation + premières impressions sur la longue traîne.
- Mois 2–3 : top 20 sur plusieurs longues traînes, montée sur les piliers.
- Mois 3–6 : top 10 sur des longues traînes, progression « belote contrée » /
  « stratégie coinche » selon le netlinking.
